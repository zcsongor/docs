#### Application Error

An error occurred with the application because it receive more traffic than the current scale settings could handle.

<br />

<Note label={false}>
  Code: <InlineCode>V1_NO_SCALED_DEPLOYMENTS</InlineCode>
</Note>

---

#### Troubleshoot

- Try increasing the scale settings on your application:
  ```
  // scale up to 10 instances
  now scale my-deployment-lwbxweoqjo.now.sh 10
  ```
- For more help try `now scale --help` or check out the [Now Scaling Documentation](/docs/v1/getting-started/scaling).
