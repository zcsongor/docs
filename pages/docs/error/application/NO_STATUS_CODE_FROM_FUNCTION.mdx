#### Application Error

The application did not respond with a status code.

<br />

<Note label={false}>
  <b>502</b>: Bad Gateway
  <br />
  Code: <InlineCode>NO_STATUS_CODE_FROM_FUNCTION</InlineCode>
</Note>

---

#### Troubleshoot

- Make sure the Serverless Function from the application responds with a status code.
- Check the application logs, which can be found at the host URL under [the `/_logs` path](/docs/v2/advanced/concepts/special-paths#logs-view), for example:
  ```
  https://my-deployment-lwbxweoqjo.now.sh/_logs
  ```
