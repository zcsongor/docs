#### Application Error

The requested Serverless Function from the application did not return a string as the body.

This is an error with the application.

<br />

<Note label={false}>
  <b>502</b>: Bad Gateway
  <br />
  Code: <InlineCode>BODY_NOT_A_STRING_FROM_FUNCTION</InlineCode>
</Note>

---

#### Troubleshoot

- Check the application logs, which can be found at the host URL under [the `/_logs` path](/docs/v2/advanced/concepts/special-paths#logs-view), for example:
  ```
  https://my-deployment-lwbxweoqjo.now.sh/_logs
  ```
- Check if the application is returning an empty response, or possibly `null`.
