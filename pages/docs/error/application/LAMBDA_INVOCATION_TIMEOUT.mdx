#### Application Error

The request for a Serverless Function reached the timeout threshold (5 minutes). This is an application error.

<br />

<Note label={false}>
  Code: <InlineCode>LAMBDA_INVOCATION_TIMEOUT</InlineCode>
</Note>

---

#### Troubleshoot

- A common cause for this issue is when the application contains an unhandled exception. Check the application logs, which can be found at the host URL under [the `/_logs` path](/docs/v2/advanced/concepts/special-paths#logs-view), for example:
  ```
  https://my-deployment-lwbxweoqjo.now.sh/_logs
  ```
